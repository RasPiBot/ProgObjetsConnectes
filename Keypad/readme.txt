Bonjour, et bienvenue a ce petit test de clavier 4x4 matriciel

Pour commancer, veuillez installer cette librairie avec la commande suivante : 

        sudo pip3 install adafruit-circuitpython-matrixkeypad

Voici les pins et la GPIO correspondant : 

pin 1 : GPIO 18
pin 2 : GPIO 23
pin 3 : GPIO 24
pin 4 : GPIO 25
pin 5 : GPIO 19 / MOSI
pin 6 : GPIO 22
pin 7 : GPIO 27
pin 8 : GPIO 17
