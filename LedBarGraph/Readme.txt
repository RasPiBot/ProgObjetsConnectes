
Présentation de la Bar Led Graph.

BARLED.py est une interface à exécuter en mode konsol sur le Raspberry pi directement.
	Commande:python3 BARLED.py

BARLED1.py peut être démarrer par ssh à distance. Allume un Led aprÈs l'autre
	Commande:python3 BARLED1.py

BARLED2.py peut être démarrer par ssh à distance.aAllumer toute les lumieres et en éteindre 1 après l'autre.
	Commande:python3 BARLED2.py

BARLED3.py peut être démarrer par ssh à distance.allumer lumière de l'extérieur vers l'intérieur..
	Commande:python3 BARLED3.py 

Voici les GPIO utilisé : 

---LED utilisé---

LED #2 - GPIO-17 
LED #3 - GPIO-27 
LED #4 - GPIO-22 
LED #5 - GPIO-5  
LED #6 - GPIO-18 
LED #7 - GPIO-23 
LED #8 - GPIO-24 
LED #9 - GPIO-12 
LED #10- GPIO-16
